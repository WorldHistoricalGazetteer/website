Dataset Collections

A WHG dataset collection logically links two or more datasets, in order to visualize and query their combined contents together. The Dataset Collection feature can also be considered a gazetteer builder tool, as explained below.

Simple linking guide

Declare multiple datasets as linked, and add an explanatory abstract, an essay, an image, and links to related web resources.

Gazetteer building guide

Often, place references are drawn from multiple sources—sometimes from multiple projects. In this case, records for the same place must be conflated—that is, reconciled with each other. Version 3 of the WHG introduces the capability to assemble a dataset collection by reconciling datasets with each other.

Place Collections

A place collection in WHG consists of individual place records drawn from multiple public datasets. Each can be annotated to explain its inclusion and to describe its relation to other places in the collection.
