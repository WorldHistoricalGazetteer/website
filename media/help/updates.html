Updating datasets

NOTE: Dataset updates are available only for LP-TSV formatted files at this time.

Datasets can be updated by uploading a new file, the contents of which will replace the previous. Each record in the new file is considered by the upload script in turn:

  • If it has a feature/row identifier ('src_id') matching one in the WHG database, it will replace the existing record. However, existing associated place_link and place_geom records that were added during previous reconciliation steps will remain unchanged.
  • If it has a feature/row identifier ('src_id') that does not exist, it will be added to the database. Such records will need to be processed as before: reconciled against Wikidata prior to accessioning (reconciliation with the WHG index).
