Teaching with the WHG

The World Historical Gazetteer project is making the data it publishes useful to teachers at secondary- and post-secondary levels with our Place Collection feature and contributed lesson plans.

Place Collection Groups guide

Any registered WHG user can create an annotated Place Collection, composed of records from any number of public datasets in the system.

New in Version 3.0 is the capability for teachers and workshop leaders to create private Place Collection groups, add registered users as members, review submitted collections, and release collections into a class/group gallery visible to its members. Teachers can also nominate exemplary collections from their group/class to display in the Student section of the site-wide Collection Gallery.

Details for how to create and manage groups—and for students or workshop participants to submit a collection to a group—are explained in this tutorial.
