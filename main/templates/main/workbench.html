Gazetteer Workbench

WHG offers four pathways for using its tools and services
to augment and publish historical place data

1 Learn more

Augment your place records by discovering missing coordinate locations (geocoding) and adding name variants.

  1. Create a place data file
  2. Upload it to your private workspace
  3. Reconcile your place records with Wikidata and GeoNames
  4. Download the augmented data for offline use
  5. Optionally publish the dataset (see Pathway 2.)

2 Learn more

Publish a single dataset built in Pathway 1 as a gazetteer resource in WHG

  1. Add metadata, an image, and essay to your augmented dataset
  2. Notify WHG editors you wish to publish it
  3. Once accepted, its public presentation will be visible to the public, and its place records accessible in WHG's search, browse, and API features
  4. A further step of linking your records with those of other datasets in the WHG index is strongly encouraged!

3 Learn more

Assemble an historical gazetteer from multiple datasets created and published in Pathways 1 and 2.

  1. Create a new, empty Dataset Collection
  2. Add published datasets to it, one-by-one. These can be your own or those of collaborators on the collection.
  3. Records for closely matched places between datasets will appear linked in the collection
  4. Complete metadata, and add an image and essay to the collection

4 Learn more

Author and publish a thematic annotated place collection

  1. Create a new, empty Place Collection
  2. Search for and add places to your collection
  3. Optionally, create a new dataset and add all its places at once
  4. Annotate each place record with a note, time information, custom keywords
  5. Add metadata, external links, and an essay–then request publication
  6. Once accepted, your collection will be visible to the public
